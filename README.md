## 💖 𝐀𝐛𝐨𝐮𝐭 𝐓𝐡𝐞 𝐏𝐫𝐨𝐣𝐞𝐜𝐭

> 🎀 *A GOAT-Bot Repo made for easy use. It can be used to make Messenger bots.* 🌨️

**Mahiru Shina** is a continuation of the GOAT-Bot legacy, redesigned specifically for Messenger. Whether you want to automate, moderate, entertain, or experiment – this bot gives you a solid foundation.

🎯 **Project Goals:**
- 🚀 Stable & feature-packed Messenger bot framework
- 👩‍💻 Easy for developers of all levels
- 🎀 Kawaii aesthetic + powerful functionality

<br>

## 🚀 𝐐𝐮𝐢𝐜𝐤 𝐒𝐭𝐚𝐫𝐭

```bash
# Clone the repo
git clone https://github.com/Dbz-Mahin7x/Mahiru-shina.git

# Enter directory
cd Mahiru-shina

# Install dependencies
npm install

# Configure (edit config.json)
# Add your Facebook credentials

# Start the bot
npm start
```
